# Simple Terrain
This is a simple terrain built using JS and GLSL code.

## Controls

O: Observer View
R: Player View

Observer View:
- Left Click + Drag: Pan
- Right Click + Drag: Orbit
- Scroll Wheel: Zoom
        
Player View:
- W/S: Move Forward/Backward
- A/D: Turn Left/Right
- Right Click + Drag: Free Look (Locks Cursor)
